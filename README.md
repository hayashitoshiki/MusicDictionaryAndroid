# MusicDictionaryAndroid
 
レコメンデータ蓄積型によるアーティスト検索アプリ

## 主な機能
* ログイン
* CRUD
 
## 言語
Kotlin

## アーキテクチャ
MVVM

## 主な使用技術
* Koin
* Retrofit
* OkHttp
* Gson
* Coroutine
* ktLint
* Firebase
* Room
* LiveData
* DataBinding
