# MusicDictionaryAndroid
 
ユーザー投稿のレコメンドデータ蓄積型によるアーティスト検索アプリ

## 主な機能
* 認証機能
* アーティスト登録・編集・削除
* アーティスト検索
 
## 言語
Kotlin

## アーキテクチャ
MVVM

## 主な使用技術
###  ネイティブ(ライブラリ)
* DI
  * Koin
* 通信
  * Retrofit
  * OkHttp
  * Kotlin Serialization
* 非同期
  * Coroutine
* DataBase
  * Room
* UI
  * LiveData
  * DataBinding
* フォーマッター
  * ktLint
* その他
  * Firebase

### バックエンド
* Ruby  
* Rails  
* SQLite(開発環境)  
* PostgreSQL(デプロイ時)  
* Heroku  
* Spotify API
