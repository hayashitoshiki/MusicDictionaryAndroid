# MusicDictionaryAndroid
 
ユーザー投稿のレコメンドデータ蓄積型によるアーティスト検索アプリ

## 主な機能
* 認証機能
* アーティスト登録・編集・削除
* アーティスト検索
 
## 言語
Kotlin

## アーキテクチャ
MVVM

## 主な使用技術
###  ネイティブ(ライブラリ)
* Koin
* Retrofit
* OkHttp
* Gson
* Coroutine
* ktLint
* Firebase
* Room
* LiveData
* DataBinding

### バックエンド
* Ruby  
* Rails  
* SQLite(開発環境)  
* PostgreSQL(デプロイ時)  
* Heroku  
* Spotify API
